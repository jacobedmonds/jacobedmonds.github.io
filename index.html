Projects

Penalty Kick Project (STOR 538)

An empirical analysis of 300 penalty kicks examining how run-up step routines relate to shot placement and outcomes. Built a fully reproducible RMarkdown workflow with spatial heatmaps and nonparametric tests.

  • Classified run-up routines using empirical quartiles (Shortened, Normal, Lengthened)
  • Visualized ball placement with goal-zone heatmaps (percentages include off-target attempts)
  • Tested step differences using Shapiro–Wilk and Wilcoxon rank-sum
  • Links: GitHub Repo • Research PDF • RMarkdown
